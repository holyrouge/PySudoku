INSTRUCTIONS

Sudoku is a number placing puzzle based on a 9x9 grid with several given numbers.

The object is to place the numbers 1 to 9 in the empty squares so that each row,

each column and each 3x3 box contains the same number only once.


In this version of the classic game, you will go through various levels of Sudoku,

starting from a 3x3 and completing each game until you reach the ultimate level, the 9x9 grid.
